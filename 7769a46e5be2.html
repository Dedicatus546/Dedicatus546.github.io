后记

如果你是本地生成静态文件之后提交到 github ，那么你完全没有必要使用这个插件，但是你就要承担仓库丢失的风险

即使你把 md 的仓库也放到了 github 上，但是重新拉取仓库会影响文件的 modifyTime 信息，导致 updated 字段错误显示

所以这个小工具只适用于开启了手动更新 updated 字段，解放每一次 commit 前需要手动更新 updated 字段的操作
